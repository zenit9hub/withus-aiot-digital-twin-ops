## 이 사이트의 목적

이 사이트는 산학 협력 멘토링 과정에서 정리된 공개용 레퍼런스 문서를 웹에서 쉽게 탐색할 수 있도록 구성한 문서 사이트입니다.

- 프로젝트 개요를 빠르게 파악
- 아키텍처와 시나리오를 단계적으로 이해
- 학생, 멘토, 협업자가 같은 기준 문서를 참조
- 향후 샘플 코드와 다이어그램을 함께 공개

## 현재 포함 범위

- 프로젝트 배경과 범위
- 공통 설계 문서 요약
- 설비환경시뮬레이터 아키텍처
- 엣지제어운영서버 아키텍처
- 공개 운영 및 협업 기준

## 원문 설계 문서

현재 사이트는 `요약 페이지 + 원문 설계 문서 링크` 구조로 시작합니다.  
원문 파일은 공개 레포 내부에서 계속 관리됩니다.
