# 프로젝트 개요

## 주제

멀티에이전트 AI 기반 IoT 디지털트윈 설비 운영지원 시스템 개발

## 목표

설비 데이터를 실시간으로 수집하고, 엣지 제어와 AI 기반 운영지원을 결합해 설비 상태를 모니터링하는 운영지원 시스템을 설계하고 구현하는 것입니다.

핵심 MVP 범위는 다음과 같습니다.

- 설비 1종 기준 시뮬레이션 또는 센서 연동
- 온도와 진동 중심의 상태 모니터링
- MQTT 기반 데이터 수집과 제어 흐름
- 엣지 서버 기반 제어와 운영지원 연계
- 운영 대시보드 기반 디지털트윈 시각화
- 멀티에이전트 AI 기반 운영지원 구조

## 핵심 개념

- 설비환경시뮬레이터: 설비 상태와 환경 변화를 생성하고 센서값을 발행
- 엣지제어운영서버: 수집, 상태 판정, 제어, 이벤트 생성, AI 연계를 담당
- 운영 대시보드: 상태, 이벤트, 운영 권고를 시각적으로 제공
- 멀티에이전트 AI: 이상진단, 운영지원, 종합 판단을 역할별로 분리

## 현재 문서 체계

- 공통 설계: PRD, WBS, 시나리오, 메시지 스키마, 에이전트 설계
- 시뮬레이터: 설비환경시뮬레이터 아키텍처
- 엣지 서버: 제어 규칙, 플로우, AI 연계, Dashboard
