# 프로젝트 개요

## 주제

AIoT 디지털 트윈 팩토리 운영지원 시스템

## 목표

공장 환경 시뮬레이터에서 발행되는 센서와 액추에이터 상태를 MQTT로 수집하고, Node-RED 기반 디지털 트윈 서버에서 인리치먼트, 룰엔진, AI 에이전트 협업, 대시보드 시각화까지 연결하는 실습형 시스템을 구현합니다.

핵심 학습 범위는 다음과 같습니다.

- Node.js 기반 공장 환경 시뮬레이터
- 온도, 진동, 컨베이어벨트, 에어컨 상태 모니터링
- `kiot/{uniq-user-id}` 기반 MQTT 토픽 설계
- Node-RED 시트 1~5 기반 디지털 트윈 처리 흐름
- 룰엔진 기반 안전 제어
- AI 에이전트 기반 현장 해석과 운영 권고
- 2D Dashboard와 3D 메타버스 팩토리 시각화

## 핵심 개념

- 공장 시뮬레이터: 실제 센서와 설비를 대신해 raw 데이터를 발행하고 제어 메시지를 수신합니다.
- 디지털 트윈 인리치먼트: raw 데이터를 판단 가능한 최신 상태 스냅샷으로 정리합니다.
- 룰엔진: 주입된 규칙에 따라 최종 제어를 수행합니다.
- AI 에이전트: 현장 분석가와 관리자 페르소나로 운영 해석과 권고를 제공합니다.
- 대시보드: 센서, 룰 결과, AI 메시지를 한 화면에서 확인합니다.
