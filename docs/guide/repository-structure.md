# 저장소 구조

## 레포 역할

이 저장소는 공개 가능한 설계 문서와 협업 자산을 관리하는 공개 레퍼런스 레포입니다.

포함되는 내용:

- 과제 개요
- 공통 설계 문서
- 시뮬레이터 아키텍처
- 엣지 서버 아키텍처
- 향후 공개 가능한 샘플 코드와 다이어그램

제외되는 내용:

- 학생 실명, 학번, 이메일
- 문의 메일 원문
- 학교 포털 캡처와 제출 흔적
- 내부 운영 메모

## 현재 폴더 구조

```text
.
├─ README.md
├─ 00_공개 운영 원칙.md
├─ docs/
└─ 멀티에이전트 AI 기반 IoT 디지털트윈 설비 운영지원 시스템 개발/
   ├─ 00_공통설계/
   ├─ 10_설비환경시뮬레이터/
   └─ 20_엣지제어운영서버/
```

## 운영 원칙

- 공개 가능한 구조, 설계, 코드만 유지
- 기존 설계 문서는 원문으로 보관
- 사이트 문서는 원문 문서를 탐색하기 쉽게 요약하고 연결
- 공개 협업은 Pull Request 중심으로 진행
