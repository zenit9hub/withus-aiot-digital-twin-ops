# WITHUS AIoT Digital Twin Ops

산학 협력 멘토링 과정에서 정리한 공개용 레퍼런스 저장소입니다.  
멀티에이전트 AI 기반 IoT 디지털트윈 설비 운영지원 시스템을 주제로, 설계 문서와 향후 샘플 구현 자산을 함께 관리합니다.

## 프로젝트 개요

이 레포는 산업 현장의 설비 데이터를 실시간으로 수집하고, 엣지 제어와 AI 기반 운영지원을 결합해 설비 상태를 모니터링하는 시스템을 다룹니다.

현재 기준 MVP(Minimum Viable Product)는 아래 범위를 중심으로 정리되어 있습니다.

- 설비 1종 기준 시뮬레이션 또는 센서 연동
- 온도, 진동 중심 상태 모니터링
- MQTT(Message Queuing Telemetry Transport) 기반 데이터 흐름
- 엣지 서버 기반 제어 및 운영지원 연계
- 운영 대시보드 기반 디지털트윈 시각화
- 멀티에이전트 AI 기반 운영지원 구조

## 공개 운영 원칙

이 저장소는 공개 협업용 레퍼런스 레포입니다.

- 공개 가능한 설계 문서와 레퍼런스 코드만 관리합니다.
- 학생 실명, 학번, 이메일, 문의 메일 원문, 학교 포털 캡처는 포함하지 않습니다.
- 새 문서는 비공개 운영 공간에서 먼저 작성한 뒤, 공개 가능 여부를 검토하고 반영합니다.

세부 기준은 [00_공개 운영 원칙.md](./00_%EA%B3%B5%EA%B0%9C%20%EC%9A%B4%EC%98%81%20%EC%9B%90%EC%B9%99.md)에서 관리합니다.

## 협업 방식

- 공개 가능한 설계 개선, 문서 정리, 샘플 구현 제안은 Pull Request로 협업합니다.
- 개인정보 또는 운영 자료가 포함된 변경은 이 레포에서 다루지 않습니다.
- 공개 레포에 반영할 내용은 가급적 구조화된 문서와 재사용 가능한 코드 중심으로 정리합니다.

## 향후 확장 방향

- Node.js 기반 설비환경시뮬레이터 구현
- Node-RED 기반 엣지 제어 플로우 구현
- 운영 대시보드 예시 구현
- 멀티에이전트 AI 운영지원 샘플 연동
